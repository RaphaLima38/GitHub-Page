# Uso do Google Gemini

O Google Gemini foi considerado como ferramenta de apoio para revisar a implantação do PortfolioHUB. A função da IA nesta entrega é auxiliar no planejamento, na conferência de boas práticas e na preparação da apresentação, sem substituir a responsabilidade do aluno sobre o conteúdo final.

## Pontos revisados com apoio da IA

- Organização das etapas de implantação.
- Boas práticas básicas de segurança para um repositório público.
- Controle de acesso usando GitHub.
- Clareza da documentação.
- Estrutura do roteiro do vídeo final.

## Prompts sugeridos

### Revisão da implantação

```text
Estou fazendo a entrega final de um PortfolioHUB publicado no GitHub Pages.
Revise se meu plano de implantação cobre Git, GitHub, publicação, segurança,
controle de acesso, testes e apresentação final.
```

### Revisão de segurança

```text
Tenho um site estático de portfólio publicado no GitHub Pages.
Quais cuidados básicos de segurança devo documentar para um repositório público,
considerando que não há login nem banco de dados?
```

### Revisão do vídeo

```text
Preciso gravar um vídeo curto apresentando a entrega final do meu PortfolioHUB.
Monte uma sequência simples para eu explicar a evolução do projeto, a implantação,
o GitHub Pages, a documentação e os testes.
```

## Observação

As respostas da IA devem ser usadas como apoio de revisão. O texto final precisa ser adaptado para o contexto real do projeto e para o nível da atividade.
