# Colaboracao e Pull Requests

## Objetivo

Registrar evidencias do uso de branch, pull request e fluxo colaborativo.

## Evidencias registradas

- branch interna: `feature/repositorio-versionado`
- pull request interno: `https://github.com/RaphaLima38/portfolioHUB/pull/1`
- fork publico de pratica: `https://github.com/RaphaLima38/Spoon-Knife`
- branch no fork: `docs/pratica-pai-ceub`
- pull request de pratica no fork: `https://github.com/octocat/Spoon-Knife/pull/40009`

## O que isso demonstra

- organizacao do trabalho em branch separada
- envio da branch para o GitHub
- abertura de pull request interno
- uso de fork de repositorio publico
- envio de modificacao para o repositorio original via pull request

## Observacao

No repositorio principal, a colaboracao foi demonstrada pelo fluxo de branch e pull request. No fork de pratica, o objetivo foi registrar o uso de fork e envio de PR para um repositorio publico de treinamento.
