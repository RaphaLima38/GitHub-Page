# Projeto - saas-amigo-secreto

## Resumo

Proposta de modelagem para um sistema SaaS de amigo secreto.

## Objetivo

Organizar um modelo que contemple:

- participantes
- edicoes do evento
- patrocinadores
- sorteios
- mensagens
- notificacoes
- avisos gerais
- sugestoes de presente
- feedback

## Pontos principais

- uso de entidade associativa para participacao em cada edicao
- separacao entre mensagens e notificacoes
- historico de sorteios por edicao
- estrutura preparada para evolucao futura

## Observacao

Este projeto esta documentado em forma de proposta de modelagem e ainda pode ser expandido com artefatos visuais e normalizacao mais detalhada.
