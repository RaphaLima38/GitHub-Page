# Projeto - modelagem-produtos-limpeza

## Resumo

Caso academico de modelagem de banco de dados para uma firma de produtos de limpeza.

## Objetivo

Representar de forma organizada:

- clientes
- categorias
- produtos
- pedidos
- itens de pedido

## Pontos principais

- identificacao das entidades centrais
- definicao de cardinalidades
- uso de chave composta em item de pedido
- preocupacao com 1FN, 2FN e 3FN

## Entregaveis

- `artefatos/DER Caso 1 - Produtos de Limpeza.pdf`

## Aplicacao

Este caso mostra uma modelagem mais direta, boa para exercicios de DER e para explicar relacionamentos basicos de negocio.
