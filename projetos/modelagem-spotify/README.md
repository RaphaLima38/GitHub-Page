# Projeto - modelagem-spotify

## Resumo

Projeto academico de modelagem de dados para um sistema inspirado em plataforma de streaming.

## Objetivo

Construir um DER mais organizado e coerente, com:

- entidades principais bem definidas
- resolucao de relacionamentos N:M
- reducao de tabelas desnecessarias
- estrutura mais clara para normalizacao

## Pontos principais

- entidades como Usuario, Plano, Assinatura, Playlist, Faixa, Album e Artista
- entidades associativas para relacoes muitos-para-muitos
- revisao de chaves primarias compostas
- foco em 1FN, 2FN e 3FN

## Entregaveis

- `artefatos/DER Spotify Corrigido - Visual.pdf`

## Base textual

Este projeto foi documentado a partir de uma revisao e simplificacao do modelo para ficar mais alinhado com atividades academicas de modelagem e normalizacao.
